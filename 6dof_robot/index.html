1. 项目概述

本项目设计了一款6-DOF（自由度）全驱动水下机器人，旨在完成水下抓取任务。与传统欠驱动水下机器人相比，6-DOF全驱动方案能够在紧凑设计的基础上实现更高水平的运动控制能力，完成复杂的水下作业任务。

1.1 设计目标

  • 实现6个自由度的独立运动控制
  • 紧凑的机械结构设计
  • 精确的目标抓取能力
  • 稳定的水下运动性能

2. 结构设计

2.1 推进器分布方案

传统推进器分布分为三类：欠驱动、全驱动和过余驱动。

类型特点适用场景
欠驱动推进器数量少于自由度数简单观测任务
全驱动推进器数量等于自由度数精细作业任务
过余驱动推进器数量多于自由度数高冗余任务

考虑到机器人需要在紧凑设计的基础上实现高水平的抓取任务，本设计选用6-DOF全驱动方案。

常规全驱动方案使用6个固定推进器或4个矢量推进器。矢量推进器虽然机动性好，但结构复杂、可靠性较低。因此，本设计提出了一种4推进器+2舵机的混合推进方案：

  • 4个主推进器提供主要推力
  • 2个舵机控制机械臂角度

2.3 材料选择

  • 主体：铝合金（轻量化和耐腐蚀）
  • 密封件：O型圈（氟橡胶）
  • 紧固件：不锈钢（耐海水腐蚀）
  • 浮力材料：EPP泡沫

3. 控制方案

3.1 运动学建模

建立机器人坐标系，使用DH参数法进行运动学建模：

关节$\theta$$d$$a$$\alpha$
1$\theta_1$$d_1$0$90°$
2$\theta_2$0$a_2$0
3$\theta_3$0$a_3$0
4$\theta_4$00$90°$
5$\theta_5$00$-90°$
6$\theta_6$$d_6$00

正运动学方程： $$T_i^{i-1} = \begin{pmatrix} \cos\theta_i & -\sin\theta_i\cos\alpha_i & \sin\theta_i\sin\alpha_i & a_i\cos\theta_i \ \sin\theta_i & \cos\theta_i\cos\alpha_i & -\cos\theta_i\sin\alpha_i & a_i\sin\theta_i \ 0 & \sin\alpha_i & \cos\alpha_i & d_i \ 0 & 0 & 0 & 1 \end{pmatrix}$$

3.2 基于深度强化学习的控制策略

为了实现复杂的抓取任务，采用深度强化学习（DRL）进行控制策略训练。

3.2.1 网络架构

采用Actor-Critic架构：

  • Actor网络：策略网络$\pi(a|s)$
  • Critic网络：价值网络$V(s)$

3.2.2 状态空间

$$s = [p_{pos}, p_{vel}, q, \dot{q}, target_{pos}, image]$$

其中：位置、速度、关节角度、目标位置、图像

3.2.3 奖励函数

$$r = r_{target} + r_{smooth} + r_{collision}$$

  • $r_{target}$：目标达成奖励
  • $r_{smooth}$：运动平滑奖励
  • $r_{collision}$：碰撞惩罚

class PIDController {
public:
    double kp, ki, kd;
    double setpoint, current_value;
    double integral = 0;
    
    double compute(double feedback) {
        double error = setpoint - feedback;
        integral += error;
        double derivative = error - last_error;
        last_error = error;
        return kp * error + ki * integral + kd * derivative;
    }
};

5.3 实物测试

  • 机器人可实现6个自由度的独立控制
  • 抓取成功率达到90%以上
  • 续航时间约2小时

6. 总结

本文设计了一款6-DOF全驱动水下机器人，主要贡献包括：

  1. 推进方案创新：提出4推进器+2舵机的混合推进方案
  2. 控制策略优化：引入深度强化学习实现自主抓取
  3. 紧凑结构设计：在有限空间内实现全驱动配置

该机器人可应用于水下探测、文物打捞、海洋生物研究等领域。

6.1 改进方向

  • 进一步优化机械臂末端精度
  • 增强视觉算法在浑水环境下的鲁棒性
  • 延长水下续航时间
