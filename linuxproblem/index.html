2. 锁定内核版本

Ubuntu 在执行 sudo apt update && sudo apt upgrade 时，默认启用了内核自动升级服务。内核升级可能会造成一些列不必要的麻烦，因此需要锁定内容后，更新当前的软件。

  1. 检查内核版本： uname -r 假设是5.15.0-67-generic
  2. 锁定当前内核版本
1
2
3
4

sudo apt-mark hold linux-image-5.15.0-67-generic
sudo apt-mark hold linux-headers-5.15.0-67-generic
sudo apt-mark hold linux-modules-5.15.0-67-generic
sudo apt-mark hold linux-modules-extra-5.15.0-67-generic
  1. 验证 输入apt-mark showhold查看是否锁定成功
  2. 更新软件 sudo apt upgrade

4. 双击图标缩小

配置：gsettings set org.gnome.shell.extensions.dash-to-dock click-action 'minimize'

复原：gsettings reset org.gnome.shell.extensions.dash-to-dock click-action

5. 调节屏幕亮度

使用xrandr来强制调节屏幕亮度

1
2
3
4
5

# 1. 确认你的显示器代号
xrandr | grep " connected" # 假设输出开头是DP-4

# 2. 调节亮度到 0.6（范围 0.0-1.0） 
xrandr --output DP-4 --brightness 0.6
