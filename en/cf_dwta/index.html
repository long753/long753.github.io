1. Research Background

In modern warfare systems, attacking key targets often requires considering cascading damage effects. When a critical node is destroyed, it may cause functional degradation or failure of other connected nodes. This phenomenon is called “Cascading Damage.” For example, destroying a communication base station may cause nearby radar stations to lose information support, thereby affecting the overall effectiveness of the air defense system.

Traditional damage probability models only consider the direct damage effect of a single strike, ignoring the relationships between targets. Therefore, establishing a damage probability model considering cascading effects has important theoretical significance and practical value.

2. Problem Definition

There are N interconnected target nodes, forming a network system $G=(V,E)$, where:

  • $V = {v_1, v_2, …, v_N}$ represents the set of nodes
  • $E \subseteq V \times V$ represents the set of edges, indicating the relationships between nodes

2.1 Basic Assumptions

  1. Each node $v_i$ has an initial functional value $F_i \in [0, 1]$, where 1 means fully functional and 0 means completely failed
  2. Each node has a failure threshold $\theta$; when the functional value falls below the threshold, it is considered failed
  3. Edge $(v_i, v_j)$ has a weight $w_{ij} \in [0, 1]$, indicating the degree of dependency between nodes

3. Mathematical Model

3.1 Direct Damage Model

Assume that after striking target $v_i$, its functional value decreases to: $$F_i’ = F_i \cdot (1 - d_i)$$

Where $d_i$ is the damage degree to the target, $d_i \in [0, 1]$.

The damage degree $d_i$ follows a probability distribution, usually described by exponential or normal distribution: $$P(d_i \leq x) = 1 - e^{-\lambda x}$$ or $$P(d_i \leq x) = \Phi\left(\frac{x - \mu}{\sigma}\right)$$

3.2 Cascading Damage Model

Considering the dependency between nodes, define the functional value of node $v_i$ as: $$F_i = f({F_j}{j \in N(i)}, {w{ij}})$$

Where $N(i)$ is the set of neighbor nodes of $v_i$.

Using the weighted average model: $$F_i = \sum_{j \in N(i)} w_{ij} \cdot F_j$$

When a node is struck and its functional value decreases, it affects neighbor nodes through the above relationship, generating cascading effects.

3.3 Damage Probability Calculation

Define the system damage probability as the probability that system functionality drops below the threshold: $$P_{damage} = P\left(\sum_{i=1}^N F_i < \theta \cdot \sum_{i=1}^N F_i^0\right)$$

Where $F_i^0$ is the initial functional value of the node.

3.4 DWTA Model

The DWTA (Damage Weapons Target Assignment) problem can be expressed as:

$$\max \sum_{i=1}^N P_i(x_i)$$ $$s.t. \sum_{i=1}^N c_i x_i \leq C$$ $$x_i \in {0, 1}$$

Where:

  • $P_i(x_i)$ is the damage probability of node i after strike
  • $c_i$ is the resource consumption required to strike node i
  • $C$ is the total resource constraint

4.1 Monte Carlo Simulation

Algorithm: Monte Carlo Simulation
Input: Network G, strike plan S
Output: Damage probability estimate P_damage

for k = 1 to M:
    for each node i in V:
        if i in S:
            d_i = Random(distribution)
            F_i = F_i^0 * (1 - d_i)
        else:
            F_i = F_i^0
    
    repeat:
        changed = false
        for each node i in V:
            F_i_new = sum(w_ij * F_j for j in N(i))
            if |F_i_new - F_i| > epsilon:
                F_i = F_i_new
                changed = true
    until not changed
    
    if sum(F_i) < theta * sum(F_i^0):
        count++
    
P_damage = count / M

4.2 Approximate Analytical Solution

For large-scale networks, mean-field approximation can be used:

Assuming node functional values follow a normal distribution $F_i \sim N(\mu_i, \sigma_i^2)$, then the system functional value: $$\sum_{i=1}^N F_i \sim N\left(\sum_{i=1}^N \mu_i, \sum_{i=1}^N \sigma_i^2 + 2\sum_{(i,j)\in E} \rho_{ij}\sigma_i\sigma_j\right)$$

Where $\rho_{ij}$ is the correlation coefficient between nodes.

5.2 Result Analysis

The simulation results show:

  1. Cascading effects can increase total damage probability by approximately 20-40%
  2. Network robustness is closely related to node degree distribution
  3. Identification of critical nodes has significant impact on damage efficiency

6. Conclusion

This paper establishes a damage probability model considering cascading effects. The main contributions include:

  1. Proposed a cascading damage propagation mechanism based on functional dependency
  2. Provided a Monte Carlo calculation method for damage probability
  3. Analyzed the influence of network topology on damage effects

This model can provide decision support for weapons target assignment and can also be used to evaluate the anti-damage capability of key targets.

6.1 Application Scenarios

  • Combat Command: Selecting optimal target combinations
  • Defense Evaluation: Evaluating anti-damage capability of critical infrastructure
  • Plan Formulation: Developing targeted defense strategies

6.2 Limitations

  1. Dependency assumptions are oversimplified
  2. Time dynamics are not considered
  3. Model parameters are difficult to obtain accurately
