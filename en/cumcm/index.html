1. Problem Background

The “Bench Dragon” (板凳龙) is a traditional Chinese folk activity where participants cooperate to swing a long dragon made of wooden boards connected by handles. This problem investigates the complete process of the dragon team spiraling inward along an Archimedean spiral, turning around along the shortest path, and spiraling outward. It involves:

  • Motion modeling in spiral space
  • Distance constraints between handles
  • Collision detection and avoidance
  • Optimization design of the turnaround path

3. Main Algorithms

3.1 Problem 1: Spiral Position Modeling

  • Based on constant-velocity Archimedean spiral motion model
  • Solve angle-position relationship using arc length integration and inverse functions
  • Calculate all handle positions recursively using the cosine theorem

3.2 Problem 2: Collision Detection

  • Rectangular collision model: Simplify handles as rectangles
  • Nearest-neighbor angle search algorithm
  • Time-series state detection

3.3 Problem 3: Turnaround Space Analysis

  • Geometric constraint analysis of turnaround region
  • Binary search for boundary conditions
  • Polynomial curve fitting

3.4 Problem 4: Turnaround Path Optimization

  • Establish double-arc turnaround model
  • Constraint handling (jam detection, angle range)
  • Numerical optimization for minimum path

3.5 Problem 5: Velocity Optimization

  • Time-stepping simulation
  • Solve speed limits under constraints
  • Binary search for optimal initial velocity

4. Environment Requirements

  • MATLAB R2018b or higher
  • Optimization Toolbox
  • Recommended memory ≥ 8GB

7. Data Files

  • theta.mat, r.mat - Angle and radius data for Problem 1
  • x_opt.mat - Optimal solution for Problem 4
  • result.mat - Calculation results for Problem 2

8. Notes

  1. Some calculations may take considerable time; please reserve sufficient computation time
  2. Optimization algorithms are sensitive to initial values; existing optimal solutions can be used as initial values
  3. Collision detection includes a safety margin
