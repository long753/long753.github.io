1. Project Overview

This project designs a 6-DOF (Degree of Freedom) fully-actuated underwater robot for underwater grasping tasks. Compared with traditional under-actuated underwater robots, the 6-DOF fully-actuated scheme can achieve higher-level motion control capability on a compact design basis, completing complex underwater operation tasks.

1.1 Design Objectives

  • Achieve independent motion control of 6 degrees of freedom
  • Compact mechanical structure design
  • Precise target grasping capability
  • Stable underwater motion performance

2. Mechanical Design

2.1 Thruster Distribution Scheme

Traditional thruster distribution is divided into three categories: under-actuated, fully-actuated, and over-actuated.

TypeCharacteristicsApplication
Under-actuatedFewer thrusters than DOFsSimple observation tasks
Fully-actuatedEqual thrusters and DOFsPrecise operation tasks
Over-actuatedMore thrusters than DOFsHigh redundancy tasks

Considering that the robot needs to achieve high-level grasping tasks on a compact design basis, this design adopts the 6-DOF fully-actuated scheme.

The conventional fully-actuated scheme uses 6 fixed thrusters or 4 vector thrusters. Although vector thrusters have better maneuverability, they have complex structure and low reliability. Therefore, this design proposes a 4-thruster + 2-servo hybrid propulsion scheme:

  • 4 main thrusters provide primary thrust
  • 2 servos control the mechanical arm angle

2.3 Material Selection

  • Main Body: Aluminum Alloy (lightweight and corrosion-resistant)
  • Seals: O-rings (Fluororubber)
  • Fasteners: Stainless Steel (seawater corrosion-resistant)
  • Buoyancy Material: EPP Foam

3.2 Deep Reinforcement Learning Based Control Strategy

To achieve complex grasping tasks, Deep Reinforcement Learning (DRL) is used for control strategy training.

3.2.1 Network Architecture

Using Actor-Critic architecture:

  • Actor Network: Policy network $\pi(a|s)$
  • Critic Network: Value network $V(s)$

3.2.2 State Space

$$s = [p_{pos}, p_{vel}, q, \dot{q}, target_{pos}, image]$$

Where: position, velocity, joint angle, target position, image

3.2.3 Reward Function

$$r = r_{target} + r_{smooth} + r_{collision}$$

  • $r_{target}$: Target achievement reward
  • $r_{smooth}$: Motion smoothness reward
  • $r_{collision}$: Collision penalty

class PIDController {
public:
    double kp, ki, kd;
    double setpoint, current_value;
    double integral = 0;
    
    double compute(double feedback) {
        double error = setpoint - feedback;
        integral += error;
        double derivative = error - last_error;
        last_error = error;
        return kp * error + ki * integral + kd * derivative;
    }
};

6. Conclusion

This paper designs a 6-DOF fully-actuated underwater robot. Main contributions include:

  1. Propulsion Scheme Innovation: Proposed 4-thruster + 2-servo hybrid propulsion scheme
  2. Control Strategy Optimization: Introduced deep reinforcement learning for autonomous grasping
  3. Compact Structure Design: Achieved fully-actuated configuration in limited space

This robot can be applied in underwater exploration, cultural relic salvage, marine biological research, and other fields.

6.1 Improvement Directions

  • Further optimize robotic arm end accuracy
  • Enhance vision algorithm robustness in turbid water environment
  • Extend underwater endurance time
