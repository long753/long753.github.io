1. Competition Overview

The National Undergraduate Intelligent Vehicle Competition is one of the college student science and technology competitions endorsed by the Ministry of Education. It aims to cultivate students’ innovative practical abilities and teamwork spirit. The 20th competition includes multiple groups such as the Full Model Group, Vision Group, and Hovercraft Group. The Full Model Group requires participating teams to design and build intelligent vehicles capable of autonomously completing track recognition, element processing, and path planning tasks.

This project is based on the EdgeBoard embedded platform, using YOLOv3-Mobilenet object detection algorithm combined with traditional image processing methods to implement a complete intelligent vehicle control system.

2. Hardware Platform

2.1 Main Control Platform

This project uses Baidu EdgeBoard as the main control platform with the following specifications:

ParameterSpecification
CPUARM Cortex-A53/A72
GPUMali-T860
Computing Power3.0 TOPS
Memory4GB LPDDR4
SystemUbuntu 18.04

EdgeBoard has a built-in PaddlePaddle inference engine, supporting deployment of various deep learning models, and has strong real-time image processing capabilities, making it very suitable for intelligent vehicle competition applications.

2.2 Sensor Configuration

  • Camera: Industrial camera for track image acquisition
  • Serial Port: USB-to-Serial converter connecting main controller and motor driver board
  • Battery: 12V lithium battery providing power for the entire vehicle

4. Key Technology Implementation

4.1 Object Detection

The YOLOv3-Mobilenet model is used for track element detection, supporting recognition of the following elements:

  • Charging Area (BATTERY)
  • Black Box Obstacle (BLOCK)
  • Bridge (BRIDGE)
  • Burger Obstacle (BURGER)
  • Traffic Cone (CONE)
  • Crosswalk (CROSSWALK)
  • Pedestrian (PEDESTRIAN)
  • Dashed Line (LINE)

void SmartCar::ai_process_mul() {
  auto ai_process_task{
    [](std::shared_ptr<Detection> detection){
      cv::Mat m;
      while(smartcar.ai_runable){
        if(!smartcar.ai_picture_update.load()) continue;
        m = std::move(smartcar.image_for_predict);
        detection->inference(m);  // Model inference
        smartcar.predict_result = detection->results;
        smartcar.ai_picture_update = false;
      }
    }
  };
  predict_thread = std::jthread(ai_process_task, this->detection);
}

void SmartCar::motion_control() {
  // Dynamic look-ahead adjustment
  if(CarParams->qianzhan_change && control.motorSpeed <= CarParams->speedHigh && control.motorSpeed >= CarParams->speedLow){
    control.qianzhan = CarParams::qianzhan_low + 
      static_cast<int>(static_cast<float>(CarParams->speedHigh - control.motorSpeed) / 
      static_cast<float>(CarParams->speedHigh - CarParams->speedLow)) * 
      (CarParams->qianzhan_high - CarParams->qianzhan_low);
  }
  
  // Send control commands via serial port
  uart->carControl(control.motorSpeed, control.servoPwm);
}

bool SmartCar::uart_connect() {
  uart = std::make_shared<Uart>("/dev/ttyUSB0");
  int ret = uart->open();
  if (ret != 0) {
    std::cerr << "Uart Open failed!" << std::endl;
    return false;
  }
  uart->startReceive();
  return true;
}

6. Summary

This project implemented an intelligent vehicle control system based on EdgeBoard, completing the following work:

  1. Hardware Integration: Completed hardware connection and debugging between EdgeBoard and sensors/actuators
  2. Software Architecture: Designed and implemented a modular software architecture for easy maintenance and extension
  3. Object Detection: Deployed YOLOv3-Mobilenet model for multi-element detection
  4. Motion Control: Implemented path tracking algorithm combining PID control and dynamic look-ahead
  5. Element Processing: Completed recognition and processing of various elements including ring, intersection, bridge, and charging area

This project won the Second Prize in the South China Division of the 20th National Undergraduate Intelligent Vehicle Competition Full Model Group.
