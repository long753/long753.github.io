1. Introduction

Spline curves are essential tools in computer graphics, robot path planning, and curve fitting. Compared to high-degree polynomials, spline curves offer better local control and numerical stability, ensuring smooth curves while reducing computational complexity.

2. Spline Curve Fundamentals

2.1 Definition

A spline curve is a smooth curve composed of polynomial segments, where each segment between two knots can be represented by a polynomial. Given a knot sequence $t_0 < t_1 < … < t_n$, a k-th degree spline curve satisfies:

  • It is a k-th degree polynomial on each interval $[t_i, t_{i+1}]$
  • It has continuous (k-1)-th derivatives throughout the entire domain

2.2 Advantages

  1. Locality: Modifying one control point only affects the local region
  2. Stability: Low-degree polynomials have better numerical stability
  3. Flexibility: Curve shape can be adjusted by changing control points
  4. Smoothness: Automatic continuity of derivatives at connection points

3. Common Spline Types

3.1 B-Spline

B-Spline (Basis Spline) is the most commonly used spline curve, with basis functions defined as:

$$B_{i,p}(t) = \frac{t - t_i}{t_{i+p} - t_i} B_{i,p-1}(t) + \frac{t_{i+p+1} - t}{t_{i+p+1} - t_{i+1}} B_{i+1,p-1}(t)$$

Where p is the spline degree.

The B-spline curve is calculated as: $$C(t) = \sum_{i=0}^n N_{i,p}(t) P_i$$

Where $N_{i,p}(t)$ is the p-th degree B-spline basis function, and $P_i$ are control points.

Properties:

  • Convex Hull: The curve lies within the convex hull of the control polygon
  • Locality: The i-th control point only affects the interval $[t_i, t_{i+p+1}]$
  • Variation Reducing: The number of intersections with any line does not exceed that with the control polygon

3.2 NURBS

NURBS (Non-Uniform Rational B-Spline) is the standard curve representation in industry:

$$C(t) = \frac{\sum_{i=0}^n R_{i,p}(t) w_i P_i}{\sum_{i=0}^n R_{i,p}(t) w_i}$$

Where the rational basis function is: $$R_{i,p}(t) = \frac{N_{i,p}(t) w_i}{\sum_{j=0}^n N_{j,p}(t) w_j}$$

Advantages:

  • Supports precise conic curve representation
  • Curve shape can be adjusted through weights
  • Unified representation of free-form curves and analytical curves

3.3 Catmull-Rom Spline

Catmull-Rom spline is an interpolating spline that passes through all control points:

$$P(t) = [t^3 \quad t^2 \quad t \quad 1] M P$$

Where $M$ is the Catmull-Rom matrix.

Characteristics:

  • Interpolation passes through all control points
  • Tangents are automatically calculated: $C’(t_i) = \frac{P_{i+1} - P_{i-1}}{2}$

3.4 Bézier Curve

Bézier curve is a special form of B-spline, defined by Bernstein basis functions:

$$B(t) = \sum_{i=0}^n B_{i,n}(t) P_i$$

Where the Bernstein basis function is: $$B_{i,n}(t) = \binom{n}{i} t^i (1-t)^{n-i}$$

// Cubic B-spline trajectory planning
class SplineTrajectory {
public:
    std::vector<Eigen::Vector3d> control_points;
    std::vector<double> times;
    
    Eigen::Vector3d position(double t) {
        // Calculate position on B-spline curve
        return evaluateSpline(t);
    }
    
    Eigen::Vector3d velocity(double t) {
        // Calculate velocity (first derivative)
        return evaluateDerivative(t, 1);
    }
    
    Eigen::Vector3d acceleration(double t) {
        // Calculate acceleration (second derivative)
        return evaluateDerivative(t, 2);
    }
};

4.2 Time-Optimal Planning

Combined with velocity/acceleration constraints, perform time-optimal planning:

$$\min T$$ $$s.t. \quad |v(t)| \leq v_{max}, \quad |a(t)| \leq a_{max}$$

Trapezoidal or S-curve velocity profiles are used for planning.

4.3 Continuity Requirements

Robot motion requires $G^2$ continuity (position, velocity, acceleration continuous):

Continuity Level Requirement
$G^0$ Position continuous
$G^1$ Tangent continuous
$G^2$ Curvature continuous

5. Numerical Computation

5.1 Parameterization Methods

Curve parameterization affects curve shape:

  1. Uniform Parameterization: $t_i = i / n$
  2. Chord Length Parameterization: $t_i = \sum_{j=1}^{i} |P_j - P_{j-1}| / L$
  3. Centripetal Parameterization: $t_i = \sum_{j=1}^{i} \sqrt{|P_j - P_{j-1}|} / L$

5.2 Curve Differentiation

The derivative of a spline curve can be obtained by differentiating the basis functions:

For B-splines, the derivative of a p-th degree spline is still a (p-1)-th degree B-spline: $$\frac{d}{dt}C(t) = \sum_{i=0}^{n-1} N_{i,p-1}(t) \cdot \frac{p}{t_{i+p+1} - t_i}(P_{i+1} - P_i)$$

5.3 Curvature Calculation

Curve curvature is defined as: $$\kappa(t) = \frac{|x’y’’ - y’x’’|}{(x’^2 + y’^2)^{3/2}}$$

7. Summary

Spline curves are core tools for robot path planning. This article summarizes:

  1. Theoretical Foundation: Mathematical definition and properties of spline curves
  2. Common Types: B-spline, NURBS, Catmull-Rom, Bézier
  3. Engineering Applications: Trajectory generation and time-optimal planning
  4. Numerical Methods: Parameterization and derivative computation

Choosing the appropriate spline type depends on the specific application:

  • Trajectory Interpolation: Catmull-Rom preferred
  • Free-form Curves: B-spline or NURBS
  • Precise Representation: NURBS (conic curves)
