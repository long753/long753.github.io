1. 引言

样条曲线（Spline）是计算机图形学、机器人路径规划、曲线拟合等领域的重要工具。相比于高次多项式，样条曲线具有更好的局部控制性和数值稳定性，能够在保证曲线光滑性的同时降低计算复杂度。

2. 样条曲线基础

2.1 定义

样条曲线是由多项式片段拼接而成的平滑曲线，每两个节点之间的曲线段可以用一个多项式表示。设节点序列为$t_0 < t_1 < … < t_n$，则k次样条曲线满足：

  • 在每个区间$[t_i, t_{i+1}]$上为k次多项式
  • 在整个定义域上具有k-1阶连续导数

2.2 优势

  1. 局部性：修改一个控制点只会影响局部区域
  2. 稳定性：低次多项式数值稳定性好
  3. 灵活性：可通过调整控制点改变曲线形状
  4. 光滑性：自动保证连接点的导数连续

3. 常见样条类型

3.1 B样条

B样条（Basis Spline）是最常用的样条曲线，其基函数定义为：

$$B_{i,p}(t) = \frac{t - t_i}{t_{i+p} - t_i} B_{i,p-1}(t) + \frac{t_{i+p+1} - t}{t_{i+p+1} - t_{i+1}} B_{i+1,p-1}(t)$$

其中p为样条次数。

B样条曲线的计算公式： $$C(t) = \sum_{i=0}^n N_{i,p}(t) P_i$$

其中$N_{i,p}(t)$为p次B样条基函数，$P_i$为控制点。

特性：

  • 凸包性：曲线位于控制多边形的凸包内
  • 局部性：第i个控制点只影响区间$[t_i, t_{i+p+1}]$
  • 变差减少性：与任意直线的交点数目不超过与控制多边形的交点数

3.2 NURBS

NURBS（Non-Uniform Rational B-Spline）是非均匀有理B样条，是工业界的标准曲线表示方法：

$$C(t) = \frac{\sum_{i=0}^n R_{i,p}(t) w_i P_i}{\sum_{i=0}^n R_{i,p}(t) w_i}$$

其中有理基函数： $$R_{i,p}(t) = \frac{N_{i,p}(t) w_i}{\sum_{j=0}^n N_{j,p}(t) w_j}$$

优势：

  • 支持精确的圆锥曲线表示
  • 可通过权重调整曲线形状
  • 统一表示自由曲线和解析曲线

3.3 Catmull-Rom样条

Catmull-Rom样条是一种插值型样条，曲线通过所有控制点：

$$P(t) = [t^3 \quad t^2 \quad t \quad 1] M P$$

其中$M$为Catmull-Rom矩阵。

特点：

  • 插值通过所有控制点
  • 切线自动计算，$C’(t_i) = \frac{P_{i+1} - P_{i-1}}{2}$

3.4 Bézier曲线

Bézier曲线是B样条的特殊形式，由 Bernstein 基函数定义：

$$B(t) = \sum_{i=0}^n B_{i,n}(t) P_i$$

其中Bernstein基函数： $$B_{i,n}(t) = \binom{n}{i} t^i (1-t)^{n-i}$$

class SplineTrajectory {
public:
    std::vector<Eigen::Vector3d> control_points;
    std::vector<double> times;
    
    Eigen::Vector3d position(double t) {
        // 计算B样条曲线上的位置
        return evaluateSpline(t);
    }
    
    Eigen::Vector3d velocity(double t) {
        // 计算速度（一阶导数）
        return evaluateDerivative(t, 1);
    }
    
    Eigen::Vector3d acceleration(double t) {
        // 计算加速度（二阶导数）
        return evaluateDerivative(t, 2);
    }
};

4.2 时间最优规划

结合速度/加速度约束，进行时间最优规划：

$$\min T$$ $$s.t. \quad |v(t)| \leq v_{max}, \quad |a(t)| \leq a_{max}$$

采用梯形速度曲线或S型速度曲线进行规划。

4.3 连续性要求

机器人运动需要满足$G^2$连续（位置、速度、加速度连续）：

连续性级别 要求
$G^0$ 位置连续
$G^1$ 切线连续
$G^2$ 曲率连续

5. 数值计算

5.1 参数化方法

曲线参数化方式影响曲线形状：

  1. 均匀参数化：$t_i = i / n$
  2. 弦长参数化：$t_i = \sum_{j=1}^{i} |P_j - P_{j-1}| / L$
  3. 向心参数化：$t_i = \sum_{j=1}^{i} \sqrt{|P_j - P_{j-1}|} / L$

5.2 曲线求导

样条曲线的导数可通过求导基函数得到：

对于B样条，p次样条的导数仍然是p-1次B样条： $$\frac{d}{dt}C(t) = \sum_{i=0}^{n-1} N_{i,p-1}(t) \cdot \frac{p}{t_{i+p+1} - t_i}(P_{i+1} - P_i)$$

5.3 曲率计算

曲线曲率定义为： $$\kappa(t) = \frac{|x’y’’ - y’x’’|}{(x’^2 + y’^2)^{3/2}}$$

control_points = [0 0; 1 2; 2 1; 3 3; 4 0];

7. 总结

样条曲线是机器人路径规划的核心工具，本文总结了：

  1. 理论基础：样条曲线的数学定义与性质
  2. 常见类型：B样条、NURBS、Catmull-Rom、Bézier
  3. 工程应用：轨迹生成与时间最优规划
  4. 数值方法：参数化与求导计算

选择合适的样条类型需要根据具体应用场景：

  • 轨迹插值：优先选择Catmull-Rom
  • 自由曲线：B样条或NURBS
  • 精确表示：NURBS（圆锥曲线）
